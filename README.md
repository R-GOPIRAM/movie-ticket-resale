# Movie Ticket Resale Platform

A Spring Boot based Movie Ticket Reselling Platform that allows users to buy and sell movie tickets securely.

## Tech Stack

* Java 21
* Spring Boot
* PostgreSQL
* Spring Data JPA
* Maven
* Git & GitHub

## Features

### User Features

* Register
* Login
* Buy Tickets
* View Booking History

### Seller Features

* Create Ticket Listings
* Manage Listings
* View Sales History

### Admin Features

* Manage Users
* Manage Listings
* Monitor Transactions
